# مشروع تقرير مواقف سيارات مسجد القطري
# Qatre Mosque Parking Report Project

تقرير زيارة استكشافية لمشروع مواقف سيارات مسجد القطري في مكة المكرمة.

## وصف المشروع / Description
هذا المشروع عبارة عن تقرير ويب تفاعلي يستعرض نتائج الزيارة الاستكشافية لموقع المواقف، متضمناً السعة الاستيعابية، المداخل والمخارج، وتوثيقاً مصوراً كاملاً للمرفق.

This project is an interactive web report showcasing the results of an exploratory visit to the parking site, including capacity, entrances/exits, and full photographic documentation of the facility.

## المميزات / Features
- تصميم متجاوب (Responsive Design) متوافق مع جميع الشاشات.
- يدعم الطباعة بصيغة A4 بشكل احترافي.
- توثيق مصور شامل للموقع.
- عرض الموقع على خرائط جوجل.
- **ملاحظات هامة:**
    1. الموقع مغلق حالياً.
    2. المواقف الخارجية المحيطة مجانية.
    3. آلية العمل سوف تكون بشكل يدوي بالكامل.

## كيفية التشغيل / How to Run
يمكنك عرض التقرير ببساطة عن طريق فتح ملف `index.html` في أي متصفح ويب، أو من خلال رابط GitHub Pages بعد النشر.

You can view the report by opening `index.html` in any web browser, or via the GitHub Pages link after deployment.
